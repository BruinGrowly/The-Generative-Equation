The LJPW differential equations are integrated using the **RK4 method**, the gold standard for smooth ODE systems.

**Why RK4?**
- 4th-order accuracy (error ∝ dt⁴)
- Stable for stiff systems
- Preserves energy conservation properties
- Explicit, no matrix inversions needed

**The Algorithm:**

```python
def rk4_step(derivatives, state, dt):
    """
    Single RK4 integration step.
    
    Args:
        derivatives: Function f(state) returning [dL/dt, dJ/dt, dP/dt, dW/dt]
        state: Current [L, J, P, W]
        dt: Time step
    
    Returns:
        New state after dt
    """
    k1 = derivatives(state)
    k2 = derivatives(state + 0.5 * dt * k1)
    k3 = derivatives(state + 0.5 * dt * k2)
    k4 = derivatives(state + dt * k3)
    
    return state + (dt / 6.0) * (k1 + 2*k2 + 2*k3 + k4)


def simulate_ljpw(initial_state, duration, dt=0.01):
    """
```
